FitTrack

A fitness tracking application for recording activities, monitoring body metrics, reviewing workout history, and managing personal fitness progress.

Project Context

About FitTrack

FitTrack focuses on monitoring and managing user activities and progress. Users can create an account, record and edit activities, search existing entries, browse workouts, update their profile, and review body and calorie-related metrics from a unified mobile interface.

Problem & Direction

The Challenge

Fitness information can become difficult to follow when workouts, activity history, body measurements, and profile details are managed separately. FitTrack brings these tasks into one state-driven application so users can record progress and review relevant metrics more consistently.

Core Features

Coding Style

MVVM + REST Architecture

The application separates data, state management, and composable screens through MVVM, while REST endpoints connect the mobile client to its PHP backend through self-contained HTTP requests.

Project Outcome

A Unified Fitness Tracking Experience

FitTrack demonstrates a complete mobile workflow backed by reactive state management and REST communication. The application connects account access, workout records, body metrics, calorie guidance, exercise browsing, and profile editing through a consistent dark interface.
